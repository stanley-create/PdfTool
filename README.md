A lightweight Python utility to extract specific pages or page ranges from PDF files. It features automatic dependency management and a user-friendly CLI interface.

## How to Use

If you want to download and run this project on another machine, please follow these steps:

### 1. Clone the Project
Open your terminal (CMD or PowerShell) and enter the following command:
```bash
git clone https://github.com/stanley-create/PdfTool.git
```
### 2. Navigate to the Directory

cd PdfTool

### 3. Run the Program
The program will automatically detect and install the required pypdf library on its first run:
```bash
python main.py
```
